# Green Value Group Public Portfolio

Public portfolio dashboard for helpers, technical contributors, institutional
partners, pilot customers, sponsors and qualified investors.

## Release

`public-showcase-v0.6.1`

This release fixes the **View project** action and has been validated so that:

- project cards render;
- the project modal opens;
- no JavaScript error is raised when a project is opened;
- the project-specific Get involved action preselects the project.

## Publish with GitHub Pages

1. Create a **public** GitHub repository named `gvg-public-portfolio`.
2. Upload every file from this folder to the repository root.
3. Commit the files to the `main` branch.
4. Open **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select `main` and `/ (root)`, then save.

The first site address will normally follow this pattern:

`https://YOUR-GITHUB-USERNAME.github.io/gvg-public-portfolio/`

## Privacy boundary

Do not upload internal finance records, private Google Drive links, cap tables,
personal data, draft token documents, unapproved investment terms or the
internal registry dashboard to this public repository.
